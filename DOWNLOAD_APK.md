## 🚀 Como Instalar (3 Passos)

### **Passo 1: Baixe o APK**

O arquivo APK está disponível em:
```
https://github.com/ajsimfibra-del/Eletricista-pro-github/releases
```

**Ou clique aqui para baixar diretamente:**
- Arquivo: `app-release.apk`
- Tamanho: ~5-8 MB

### **Passo 2: Copie para seu Celular**

Opção A - Via USB:
1. Conecte seu celular ao PC via USB
2. Copie o arquivo `app-release.apk`
3. Cole na pasta `Downloads` do celular

Opção B - Via WhatsApp:
1. Envie o arquivo para você mesmo no WhatsApp
2. Baixe no celular

### **Passo 3: Instale**

1. Abra o gerenciador de arquivos do seu celular
2. Navegue até `Downloads`
3. Clique no arquivo `app-release.apk`
4. Clique em **Instalar**
5. Pronto! ✅

---

## ⚠️ Se Pedir Permissão

Se aparecer mensagem: **"Não é possível instalar apps de fontes desconhecidas"**

1. Clique em **Configurações**
2. Vá para **Apps** ou **Aplicativos**
3. Procure por **Gerenciador de Arquivos**
4. Clique em **Permissões**
5. Ative **"Instalar apps desconhecidos"**
6. Volte e tente instalar novamente
